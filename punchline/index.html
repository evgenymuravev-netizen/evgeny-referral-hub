Method, in one breath

Constrained briefs block recitation. Blind pairs kill brand bias; a formatting normalizer strips each writer's tells. All models run at provider-default decoding — several current frontier models reject manual sampling parameters, so “fixed temperature” is no longer an available protocol; the decoding regime is recorded per entry instead, and each entry carries a provenance line shown at reveal. Scores are Bradley-Terry fits with bootstrap confidence intervals. Human anchors (PRD §4) are not yet sourced, so the vs-Human column ships with the pipeline, not this page.
