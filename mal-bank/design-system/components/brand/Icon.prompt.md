**Icon** — thin-line brand icon (Lucide-style, 1.6px rounded stroke). Use anywhere Mal shows a glyph: nav, buttons, list rows, feature marks.

```jsx
<Icon name="shield-check" size={24} />
<Icon name="arrow-right" size={18} color="var(--text-muted)" />
```

Names: arrow-right, arrow-up-right, chevron-right/down, plus, check, x, search, sparkles, credit-card, shield-check, crown, message, map-pin, wallet, trending-up, arrow-down-left, bell, settings, home, user, send, eye, eye-off, info, alert-triangle. `strokeWidth` defaults to 1.6 to match the brand; keep it thin.
