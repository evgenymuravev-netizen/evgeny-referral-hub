**IconButton** — icon-only action, circular by default. Use for toolbar controls, close buttons, and app-bar affordances.

```jsx
<IconButton aria-label="Notifications"><Icon name="bell" /></IconButton>
<IconButton variant="primary" aria-label="Add"><Icon name="plus" /></IconButton>
```

Variants: `primary` | `secondary` | `ghost`. Sizes: `sm` (36) | `md` (44) | `lg` (52). `shape`: `circle` (default) | `rounded`. Always pass `aria-label`.
